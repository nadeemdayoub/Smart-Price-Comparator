# Smart Price Comparator SaaS

**AI-powered procurement intelligence for smarter purchasing decisions.**

---

## Overview

Smart Price Comparator streamlines the quotation review process using AI-powered product matching and multi-currency normalization. Upload PDFs or Excel files, let the system extract and match line items, review suggestions, and make confident purchasing decisions — all in one platform.

## Key Features

| Feature | Description |
|---------|-------------|
| **AI-Powered Extraction** | Automatically extract line items from PDF and Excel quotations |
| **Smart Product Matching** | AI matches supplier items to your canonical product catalog with confidence scores |
| **Multi-Currency Support** | Normalize prices across different currencies with configurable exchange rates |
| **Side-by-Side Comparison** | Compare supplier prices for identical products across multiple vendors |
| **Price Intelligence** | Track historical price movements and identify supplier trends |
| **Review Workflow** | Approve, reject, or create new products during the review process |

## Getting Started

### Prerequisites

- Node.js 18+
- Gemini API key (for AI extraction)

### Installation

```bash
# Install dependencies
npm install

# Set your environment variable
export GEMINI_API_KEY="your-api-key-here"

# Start development server
npm run dev
```

> The app will be available at `http://localhost:3000`

### Production Build

```bash
npm run build
```
